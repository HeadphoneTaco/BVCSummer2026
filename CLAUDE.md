## Arcade Jam — CGDA Arcade Jam 2026
- **Event:** CGDA Arcade Jam 2026, hosted on itch.io (week-long "slow jam")
- **Signed up by Sean** for the whole class; counts toward GAME 1501
- **Hard requirements** (games run on real physical arcade cabinets):
  - Windows build, fullscreen 1920×1080
  - Mouse 3 = instant quit
  - Auto-quit after 3 min of no player input
  - Arcade controller handles ALL input (no mouse/unmapped keys — causes soft-lock)
  - Capsule image required; background image optional
  - Retro resolutions: 960×540, 640×360, or 480×270 (stretch to fill)
  - Post-jam debugging period available for polish
